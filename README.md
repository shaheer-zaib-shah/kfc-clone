# Foodie — KFC Clone UI

A KFC Pakistan-inspired food ordering UI built with vanilla HTML, CSS, and JavaScript. Features an auto-advancing image carousel with navigation controls, dynamic item popups with individual product data, responsive CSS Grid layouts, and a fully styled footer.

> Built as a front-end practice project (first HTML/CSS/JS project).

## Features
- Auto-advancing banner carousel with prev/next arrows and dot indicators
- Dynamic popups for each menu item with individual name, description, and price
- Responsive CSS Grid layout for menu sections
- Hover transitions on images and navigation links
- Social media links in footer

## Tech Stack
HTML · CSS · Vanilla JavaScript
